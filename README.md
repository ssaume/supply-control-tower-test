- 台灣、中國、泰國共18座工廠

- 各產品接單LT

- 業務與所屬部門

- 產能資源名稱與每日可用能力

## 初始清空的非主檔資料

- 需求事件
- 接單LT審批與需求變更事件
- 內部SO與出貨資料
- 庫存、在製工單與需求分配
- 生產工單、委外工單與採購PO
- 有限／無限模擬結果
- Commit與跨部門決策
- 執行追蹤與Highlight
- 每日生產排程與Loading
- 供應關係的當期需求量、供給量及風險

產區現況在沒有需求時，所有工廠Loading均為0%，時間網格顯示「尚無排程」。供應網路仍保留主檔關係，但當期需求與供給顯示0或「尚無需求」。

## 資訊磚取消聚焦

在需求總覽或出貨總覽展開履行明細後：

1. 點擊工單或採購PO，系統會Highlight同一BOM分支並淡化其他項目。
2. 點擊展開區內沒有按鈕或資訊磚的空白位置，即可清除聚焦並恢復全部項目。
3. 生產供應途程的站點生態圖也支援點擊空白處清除節點聚焦。

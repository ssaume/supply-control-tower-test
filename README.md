# Supply & Commitment Control Tower — RESOURCE11

Build: `2026.07.14-RESOURCE11`

## 本版重點

- 系統維持零起點：新增需求不會自動出現庫存或在製工單。
- 新增「資源管理」群組：工單匯入、工時管理、產能管理。
- 工單只有經檔案匯入或人工KEY單後才成為交易資料。
- 每個成品的六段生產途程都有單件工時主檔。
- 工單需求工時＝工單數量 × 單件工時。
- 每個工廠資源以日為單位維護可用／不可用及可用工時。
- 產區現況直接由工單、單件工時與資源行事曆計算。
- 有限／無限模擬也改用單件工時與每日可用產能，不再使用隨機滿足率。
- 需求供給區改為前置時間估算；只有明確關聯需求的匯入工單才可列入在製供給。
